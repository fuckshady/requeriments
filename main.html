Requerimientos Funcionales

Ventas

RF-01:El sistema debe permitir crear cotizaciones con productos, cantidades, precios e impuestos.

Compras

RF-06:El sistema debe registrar proveedores y asociarlos con productos y condiciones de compra.

Inventario

RF-12:El sistema debe notificar automáticamente cuando el stock de un producto esté por debajo del mínimo.

Contabilidad

RF-15:El sistema debe generar reportes contables como balances generales, estado de resultados y libros oficiales.

CRM

RF-16:El sistema debe permitir crear leads desde formularios web o ingreso manual.

Recursos Humanos

RF-21:El sistema debe permitir generar la nómina mensual de empleados con base en horas trabajadas y deducciones.

Proyectos

RF-22:El sistema debe permitir crear proyectos, tareas y asignar responsables.

Marketing

RF-25:El sistema debe permitir crear campañas de email dirigidas a grupos de clientes.

RF-27:El sistema debe permitir enviar campañas de SMS a contactos segmentados.

Punto de Venta (PdV)

RF-30:El sistema debe sincronizar los datos de ventas con el inventario general automáticamente.
